Detailed Description

template<typename T>
class ca::opt::TransportationSimplex< T >

Transportation Simplex.

Solves minimization tranportation problem using streamlined simplex method.

Note
Corresponding maximization problem could be solved by substituting const matrix with \(C_{ij}=M-C_{ij}\) where \(M\) is a maximum cost in input matrix.
