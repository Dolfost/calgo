Detailed Description

template<typename T, typename I = T>
class ca::num::NystromFredholmSecondKind< T, I >

Solves Fredholm equation of second kind using Nystrom method.

\begin{equation} \phi(x)=f(x) + \int\limits_a^b K(x,y)\phi(y)\mathrm dy, \end{equation}

  • \(\phi(x)\) is subject function, get via operator()(value_type) or phi(value_type)
  • \(a, b\) are integrtation limits, deduced from passed nodes (via setNodes(nodes_type*))
  • \(K(x, y)\) is a kernel, and can be set via setK(K)
  • \(f(x)\) is a free unit, and can be set via setF(F)
Note
For some reason the kernel should be passed as \(-1\cdot K(x, y)\)
