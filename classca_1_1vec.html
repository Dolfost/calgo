Detailed Description

template<typename T>
class ca::vec< T >

Array class.

This class wraps vec_view and defines memory management routines. It owns it's data vec::m_vec, so it will be freed when object is destroyed.

Resize array.

Elements of old array that fit into new one will be copied to it if copy=true. If old array if smaller than new, rest of the new elements will be initialized to 0. If copy if false, then value of each element is undefined.
