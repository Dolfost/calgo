Detailed Description

template<typename T>
class ca::num::runge_kutta< T >

Runge-Kutta method for Cauchy poblem (3th order)

Butcher tableau is given in form:

\[ \alpha_1 = 0,\ \alpha_2 = \frac{2}{5},\ \alpha_3 = 1,\\ \beta_{21} = \frac{2}{5},\ \beta_{31} = -\frac{7}{8},\ \beta_{32} = 1\frac{7}{8}, \beta_{33} = 0,\\ p_1 = \frac{1}{12},\ p_2 = \frac{25}{36},\ p_3 = \frac{2}{9}. \]

Problem is given in form

\[ \frac{\mathrm d u}{\mathrm d t}=f(t,u(t)),\quad u(t_0) = u_0,\quad t_0\leq t \leq t_1.\]
