Detailed Description

template<typename T>
class ca::num::euler_cauchy< T >

Euler-Cauchy method for Cauchy poblem.

Problem is given in form

\[ \frac{\mathrm d u}{\mathrm d t}=f(t,u(t)),\quad u(t_0) = u_0,\quad t_0\leq t \leq t_1.\]
