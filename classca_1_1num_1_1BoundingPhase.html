Detailed Description

template<typename T>
class ca::num::BoundingPhase< T >

Bounding phase method.

This implementation takes functor \(f(x)\) and tries to isolate minima in some inerval \([a,b]\). It is given the starting point and starting stem size, then it determines in wich direction it should move, increasing the step by \(2^i\), where \(i\) is an iteration index.
