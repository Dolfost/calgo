Detailed Description

template<typename T>
class ca::num::bvp_galerkin< T >

Boundary value problem solution using Galerkin method Problem is given in form.

\[ (k(x)u')' - q(x)u = -f(x),\ a<x<b,\\ u(a) = \mu_1,\\ u(b) = \mu_2, \]

where \(a=0,\ b=1\) and \(q(x)=0\), \(u(a)=u(a)=0\).
